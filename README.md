# liuyanban
php留言板，css没有优化，只是随便写的
一个简单的php留言系统，可以快速留言，具有管理员管理功能。
css方面没有进行优化，这是做出了看看，比较难看
这是刚学php的时候跟着敲代码完成的，利用了php+mysql进行操作
m文件夹为手机端文件夹，里面的问题不可直接用父文件夹的文件替换，因为做了部分修改，注意路径。
